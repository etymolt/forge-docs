# Etymolt Forge — Public Documentation

Forge is post-name automation. After Etymolt verifies your brand name is clean, Forge does everything else in 48 hours: domain, DNS, email, brand kit, USPTO TM filing prep, 6-platform brand handle claims, and (Pro) onchain ENS + Farcaster identity.

## What Forge does NOT do

We do not submit signup forms, solve CAPTCHAs, sign legal agreements, or impersonate you. We provide every input you need; you click the buttons. See [BUREAU_MODEL.md](./BUREAU_MODEL.md).
